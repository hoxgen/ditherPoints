# ditherPoints
Dithering tool (Openframeworks)

## Usage:

### Image  / Video mode

mode = 0; // single image mode
outputs single images

mode = 1; // video mode
outputs png sequence

nBatch = 80;
defines how many images should be processed for a video frame

/data/src/
- folder with source files

/data/export/
- output folder
